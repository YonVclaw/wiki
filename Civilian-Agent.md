The Civilian Agent function allow you transform a AI civilian unit in to a agent. What the function does it that it removes the unint and recreates it as a agent.


```cpp
/* Arguments:
 * 0: Unit <OBJECT>
 */

[this] call cScripts_fnc_makeAgent;
```
